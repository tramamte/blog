# Image dump

```sh
$ docker save -o dump_file_name.tar IMAGE
```

```sh
$ docker load -i dump_file_name.tar
```

# Container dump

```sh
$ docker export CONTAINER_NAME|CONTAINER_ID > dump_file_name.tar
```

```sh
$ docker import dump_file_name.tar
```
